> A package that converts Postman API collection v2.1 into a proper documentated PDF file

## Install

```sh
npm install
```

## Usage

```sh
    var documentor=require("postman-documentor");
    let filepath="collection.json";
    documentor(filepath);
```

## Run tests

```sh
npm run test
```
